Template PPDB Offline - SMK Maju Jaya (XAMPP)
--------------------------------------------
Cara pakai (Windows + XAMPP):
1. Pastikan XAMPP sudah terinstall dan Apache & MySQL sudah dijalankan.
2. Ekstrak folder 'ppdb_smk_maju_jaya' ke: C:\xampp\htdocs\
   => jadi path contohnya: C:\xampp\htdocs\ppdb_smk_maju_jaya
3. Buka browser dan akses: http://localhost/ppdb_smk_maju_jaya/
4. Formulir pendaftaran akan menyimpan data ke file: data/submissions.json
5. Login Admin:
   - Username: admin
   - Password: admin123
6. Jika ingin menghapus semua data pendaftar, hapus file: data/submissions.json
7. Untuk keamanan, gunakan template ini hanya untuk tugas / demo lokal.
